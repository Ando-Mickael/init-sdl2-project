# init-sdl2-project

```bash

# Utiliser cette commande pour compiler `main.c`
gcc -o main src/main.c -Iinclude -Llib -lSDL2 -lSDL2_ttf

# Utiliser cette commande pour compiler `hello.c`
gcc -o main src/hello.c -Iinclude -Llib -lSDL2 -lSDL2_ttf

# Utiliser cette commande pour lancer votre application
.\main
```
